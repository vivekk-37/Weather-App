# Weather App

This is a frontend weather dashboard built using **HTML and CSS**.  
The project focuses on clean UI design and a structured layout that can later be connected to a real weather API.

---

## About the Project

The Weather App displays:
- Current weather information
- Hourly weather forecast
- Air condition details
- A 7-day weather forecast layout

This project is designed as a **frontend-first application**, making it easy to integrate with APIs or backend frameworks in the future.

---

## Features

- Simple and clean user interface
- Sidebar navigation menu
- Current temperature and location display
- Hourly forecast section
- Air conditions (Real Feel, Wind, Rain Chance, UV Index)
- Weekly weather forecast layout
- Well-structured HTML and CSS

---

## Technologies Used

- HTML5  
- CSS3 (Flexbox and Grid)

No frameworks or libraries are used, keeping the project lightweight and easy to understand.
